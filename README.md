# Huffman & LZW Algorithms Visualizer

An interactive web-based tool that visualizes data compression algorithms step by step using HTML, CSS, JavaScript, and Mermaid.js.

The project demonstrates how classical compression algorithms work in real time with dynamic trees, trace tables, and live mathematical statistics.

## 🚀 Features

- 🌳 Dynamic Huffman tree generation using Mermaid.js
- 📋 Step-by-step execution trace table for LZW
- 📊 Live compression statistics (bits and codes savings)
- 🧮 Mathematical formulas breakdown for savings calculation
- 🇹🇷 Turkish interface optimized for academic environments

## 🧠 Algorithms

### 🔵 Huffman Encoding
A statistical data compression algorithm that assigns variable-length codes to characters based on their frequency. It builds a binary tree from the bottom up, assigning the shortest codes to the most frequent characters.

### 🔴 LZW (Lempel-Ziv-Welch)
A dictionary-based compression algorithm that builds a translation table of strings encountered in the input data on the fly. It replaces repeated patterns with single codes, learning as it reads.
